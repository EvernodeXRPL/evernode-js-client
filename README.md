# Evernode js client
Javascript client library for Evernode.
(Only tested on NodeJS)

## Prerequisites
```
npm i -g @vercel/ncc
```

## Publish to npm
```
npm install
npm login
npm run publish
```

## Running tests
```
cd test
npm install
node test.js
```
